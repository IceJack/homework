# 테이블

| 컬럼1       |   컬럼2   |       컬럼3 |
| :---------- | :-------: | ----------: |
| 왼쪽정렬    | 중앙정렬  |  오른쪽정렬 |
| 위하        |   치킨    |        쯔앗 |
| **테스트1** | _테스트2_ | ~~테스트3~~ |

안에서 마크다운을 이용 할 수 있다.

```
| 컬럼1 | 컬럼2 | 컬럼3 |
| :------- | :-------: | -------: |
| 왼쪽정렬 | 중앙정렬 | 오른쪽정렬 |
| 위하 | 치킨 | 쯔앗 |
| **테스트1** | *테스트2* | ~~테스트3~~ |
```
